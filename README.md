# Cryptocurrency Tracker

A web application that tracks the prices of cryptocurrencies in real-time using the CoinMarketCap API. This app displays a list of cryptocurrencies with their prices, market caps, and 24-hour changes, and visualizes their prices and trends through interactive graphs.

## Features

- Real-time cryptocurrency price tracking.
- Option to select the number of cryptocurrencies to display (Top 10, 25, or 50).
- Interactive bar chart showing the current prices of selected cryptocurrencies.
- Line graph displaying the 24-hour price trend for each cryptocurrency.
- Automatic data refresh every minute.
- Error handling for API failures.

## Usage 

- Run the Dash application:
- python app.py
- Open your web browser and navigate to http://127.0.0.1:8050/ to view the application.
